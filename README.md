<h1>About La La's Slots</h1>
For a full description of the game and how it works, please read the <a href='slots documentation.docx'>slots documentation.docx file</a> in this repository.

<h2>Controls</h2>
<ul>
  <li><b>Add Money </b>(Standard) [k]</li>
  <li><b>SPIN!</b> [spacebar]</li>
  <li><b>Add Money</b> (Debug) [s]</li>
  <li><b>Toggle Autospin</b> [a]</li>
  <li><b>Open Debug Menu</b> [m]</li>
</ul>

<h2>Gameplay</h2>
Press SPIN and watch the reels go by. For full information on types of wins, see the slots documentation .docx file.
<b>This game is designed for a PORTRAIT 1920x1080P screen</b>
<p style='color:red'>&#128308; Note: This game does not take real money - please do not program it to do so</p>
